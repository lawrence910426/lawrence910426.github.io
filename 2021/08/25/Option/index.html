選擇權價格的暗示

發表於 2021-08-25

利用「選擇權價格」逆推「未來的價格」

我們的目標是利用選擇權價格，去反推出到期日標的價格落在區間內機率有多高。

簡單來說，就是用選擇權來預測未來，利用群眾的智慧，來預測未來的價格。

簡介選擇權

選擇權是一張合約，合約的規格如下

想像一下，乙想要跟甲簽訂這份契約，甲身上會被 seller 指標追蹤，乙身上會被 buyer 指標追蹤，那乙應該給甲多少錢，甲才願意跟乙簽約呢？

這個問題，每天都在期貨交易所中發生，而給出的價錢，就是這張「選擇權」的「價格」。

很快的總結一下，選擇權的價格就是「簽訂這份契約，buyer 要給 seller 的補償金額」。

還是看不懂的話，可以去看這篇文章，他對選擇權有滿詳盡的解說。

價格機率密度函數

首先，我們引入一個函數 FFF，F(x)F(x)F(x) 的意思是「在到期日（Expiration_Date）的時候，*Target_Asset 剛好等於 xxx 的機率有多少？」

這個定義有點抽象，讓我們來舉個例子好了。

佐助最近學會了血輪眼，所以佐助看穿了未來，他發現在到期日當下，價格一定是 179871798717987，所以佐助認為 FFF 有以下特性

F(x)=0,∀x≠17987F(x) = 0, \forall x \neq 17987 F(x)=0,∀x​=17987

F(x)=1,x=17987F(x) = 1, x = 17987 F(x)=1,x=17987

而妙禪藉由通靈，得知在到期日，有一半的機會價格會比 175001750017500 高，所以妙禪認為 FFF 有以下特性

∫17500∞F(x)dx=12\int_{17500}^{\infty} F(x) dx = \frac{1}{2} ∫17500∞​F(x)dx=21​

這時候，我們知道 FFF 是一個機率密度函數，而且，大家對 FFF 的看法不一樣。此時，我們想像市場是一個巨人，匯集了眾人的智慧，而市場本人也對 FFF 有自己的看法。

利用市場價格反推

這一段會有比較多的數學，本段小結會有推論的最終產物，如果你對數學有恐懼症，你可以只看這一段的小結就好。

數學建模

假定一個 Call 選擇權的價格是 PPP，而且 Strike_Price 稱作 EEE，我們會有以下等式

P=∫E∞(x−E)F(x)dxP = \int_{E}^{\infty} (x - E) F(x) dx P=∫E∞​(x−E)F(x)dx

市場這個巨人，它跟佐助還有妙禪一樣，市場也有一個他認為的 FFF。而且 FFF 必定跟 PPP 匹配，如果不匹配，那麼要嘛是 PPP 太高（太貴），要嘛是 PPP 太低（太便宜），不論是哪種發生，都不會有人簽訂合約，進而導致沒有成交，沒有成交，就不會有價格，因此，在成交量足夠高的前提下，我們知道 FFF 必然與 PPP 匹配。

為了方便起見，我們忽略所有折現率的影響。
